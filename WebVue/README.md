# platform-manager

> 瑞祥集团项目框架

## 安装步骤

- 下载依赖包

> 尽量都用 `cnpm` 安装 防止丢包
[cnpm安装步骤](http://npm.taobao.org/)

```sh
cnpm install node-sass -g  // 如果安装过不用安装
cnpm install
```

- 下载公共插件(下载到`src`目录下)

```sh
https://gitee.com
```

- 开发环境

```sh
npm run dev
```

- 打包环境  
**打包之后的文件在`dist`目录下**

```sh
npm run build      // 打包正式
npm run build:dev  // 打包测试
```

- 测试环境  
**打包之后会自动运行 `npm run test`**

```sh
npm run test // 打包之后测试
```

- vsCode 插件支持

```sh
ESLint Vetur EditorConfig
```
